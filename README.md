# Atelier de Legado

Atelier boutique especializado en la creación de libros de legado personal de alta gama. Combinamos la curatía artesanal con tecnologías de inteligencia artificial para preservar historias familiares y archivos históricos.

## ℹ️ Información del Proyecto
Este proyecto es la plataforma digital de **Atelier de Legado**, diseñada para ofrecer una experiencia de usuario sobria, minimalista y profesional, alineada con la estética de la marca en Viña del Mar, Chile.

## 🛠️ Tecnologías Utilizadas
La aplicación está construida con un stack moderno de alto rendimiento:
- **Frontend**: React con TypeScript.
- **Build Tool**: Vite.
- **Estilos**: Tailwind CSS.
- **Componentes**: shadcn-ui.
